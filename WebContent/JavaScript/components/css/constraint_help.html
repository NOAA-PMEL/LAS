You can contstrain the domain and range of the plot by doing one of the following:
  • Click, hold and drag with your mouse over the plot. N.B. Drawing on the plot does not work for older versions of Internet Explorer.
  • Type values into the text boxes for one or both of the domain and range of the plot.
Once you have constrained the plot as you would like, click the "Update Plot" button to see the new plot.
