Simple user interface examples demonstrating the use of LAS javascript components.

Hawaii time series
Demonstrates the use of the LASRequest component and how to make requests for products that appear in a new window.
Pacific ocean temps
Demonstrates the use of the LASRequest and DateWidget components and utilizes image streaming to have the requested image product returned directly to the web page client.
Please see the javascript component documentation for more information on using the LAS components for creating simple user interfaces.
